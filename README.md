https://narae-tech.github.io/mini_shopping_game/

## 프로젝트 주제

- 데이터를 정해진 기준으로 분류하여 화면에 나타내는 웹페이지

<br>

## 프로젝트 참여 인원

- 1인(토이 프로젝트)

<br>

## 프로젝트 기간

- 2021.10.14 ~ 2021.10.15

<br>

## 개요

- vanilla javascript 학습을 위한 프로젝트로 정해진 기준으로 옷(데이터)을 분류하여 화면에 나타내는 웹페이지 제작

<br>

## 프로젝트 구성 안내(기술 스택)

- html5
- css3
- javascript(vanilla js)

<br>

## 담당 역할

프론트엔드(HTML, CSS, Javascript 이용)

<br>

## 느낀점

- .json 타입의 데이터 활용법 및 html의 태그 특성 활용법 학습
- css 및 javascript 사용 능력 향상
  - css 변수 사용법 터득
  - vanilla javascript 실행 원리 및 fetch문을 통해 데이터 받아오는 방법 학습

<br>

## 버전
